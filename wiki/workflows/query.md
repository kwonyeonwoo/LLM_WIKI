# Query Workflow

위키를 기준으로 질문에 답하는 절차다.

## Steps

1. `index.md`를 읽고 관련 페이지 후보를 찾는다.
2. 관련 `wiki/` 페이지를 읽는다.
3. 필요한 경우 관련 `sources/` 노트를 확인한다.
4. 답변에는 확실한 내용과 추론을 구분한다.
5. 재사용 가치가 높은 답변은 새 분석 페이지로 저장한다.
6. 저장했다면 `index.md`와 `log.md`를 갱신한다.

## Answer Style

- 먼저 결론을 말한다.
- 근거가 약한 부분은 약하다고 표시한다.
- 출처가 있는 내용과 해석을 분리한다.
- 이미 위키에 있는 말을 그대로 반복하지 말고, 질문에 맞게 합성한다.

## Save Criteria

다음 중 하나에 해당하면 답변을 `wiki/analyses/`에 저장할 만하다.

- 여러 페이지를 종합한 비교
- 나중에 다시 사용할 가능성이 높은 설명
- 의사결정 근거
- 반론과 한계를 정리한 분석
- 새 자료 조사 방향
